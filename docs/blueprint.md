# **App Name**: HOPE Assistant

## Core Features:

- Multi-Interface Continuity: Maintains context and user memory across different interfaces (text, voice, desktop, mobile, web) to provide a consistent experience.
- Adaptive Tone: Adapts the communication style (technical, casual, emotional) based on the identified user intent.
- Memory Layers Management: Efficiently utilizes short-term, long-term user, and vector context memory to enhance the relevance and continuity of interactions. A tool will be employed for selecting whether and how to incorporate historical information into its responses.
- Task Execution Model: Executes user requests by understanding intent, clarifying ambiguities, and deciding whether to inform, guide, plan, generate, or reflect, providing concise and accurate responses.
- Failure and Uncertainty Handling: Handles situations where it's unsure by acknowledging the uncertainty, explaining what is known, and asking a clear clarifying question to avoid guessing.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6) for a sense of trust and intelligence.
- Background color: Light grey (#F0F4F7), providing a neutral and unobtrusive backdrop.
- Accent color: Soft green (#A5D6A7) to highlight important information or actions without being overwhelming.
- Font pairing: 'Inter' for body text (sans-serif) to ensure readability and 'Literata' for headings (serif) to convey intellect.
- Structured output with clear headings, bullet points, and step-by-step instructions to enhance clarity and reduce cognitive load.
